# Code test

- Backend is written in Java using Spring as framework.
- Frontend is written in TypeScript using React as framework. It's based on create-react-app, so it contains some extra bloat.

## Instructions

### Prerequisites

- Java 17
- Node

### Running Backend

1. `cd backend`
2. `./mvnw spring-boot:run`

### Testing Backend

1. `cd backend`
2. `./mvnw test`

### Running Frontend

1. `cd frontend`
2. `npm i && npm start`
